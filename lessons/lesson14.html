课文 - Texte

北京的四合院 (617 caracteres)

四合院，是中国华北地区民居中的一种组合建筑形式。所谓四合，"四"指东、西、南、北四面，"合"就是四面房屋围在一起，中间形成一个方形的院子。四合院在中国汉族民居中历史最悠久，分布最广泛。不过，只要人们一提到四合院，便自然会想到北京四合院，这是因为传统的北京四合院都有一套固定的样式，十分具有代表性，在各种各样的四合院中，北京四合院可以代表其主要特点。

北京有各种规模的四合院。最简单的四合院只有一个院子，比较复杂的有两三个，而有钱人家的，通常是由好几座四合院并列组成的。大门一般开在东南角或西北角，院中的北房是正房，比其他房屋的规模大，一般包括长辈的卧室和具备日常起居、接待客人等功能的客厅。院子的两边是东西厢房，是晚辈们生活的地方。在正房和厢房之间建有走廊，可以供人行走和休息。院子是十分理想的室外生活空间。有的人家喜欢种草、养花、种竹子，有的人家则喜欢用大盆养金鱼。院子不仅拉近了人与自然的关系，也让家里人在此得到了感情的交流，对创造生活情趣起了很大作用，因而最为人们所喜爱。

只要关闭起大门，四合院内便形成一个封闭式的小环境。住在四合院里的人不常与周围的邻居打交道。在小院里，一家人过着与世无争的日子，充分享受家庭的乐趣，自然有一种令人感到自在亲切的气氛。但也有多户合住一座四合院的情况，被称为"大杂院"，住户多为普通劳动人民。邻里之间有时虽然也有矛盾，但更多时候是互帮互助，不是亲人胜过亲人，这种浓浓的情感是许多老北京人无法忘记的。

— 改编自百度文库《建筑欣赏之四合院》

语法 - Grammaire

4 points de grammaire essentiels

1
所谓
suǒwèi
Adjectif / présentatif. Emploi 1 : "ce qu'on appelle, le prétendu" — introduit un terme qui va être expliqué ou défini. Emploi 2 : désigne ce que certaines personnes disent — souvent avec désaccord ou ironie (les soi-disant…).
所谓四合，"四"指东、西、南、北四面，"合"就是四面房屋围在一起……
(1) Ce qu'on appelle sìhé : "四" désigne les quatre côtés est, ouest, sud et nord ; "合" signifie que les maisons des quatre côtés sont réunies ensemble…
很多时候，烦恼是自己找来的，这就是所谓的"自寻烦恼"。
(1) Souvent, les ennuis sont de notre propre fait — c'est ce qu'on appelle "se chercher des ennuis soi-même".
他所谓的"新闻"，其实我们早就知道了！
(2) Sa soi-disante "nouvelle", on la connaissait depuis longtemps !
现在市场上所谓的"健康食品"其实没有统一的标准。
(2) Les soi-disant "aliments santé" sur le marché actuel n'ont en réalité aucun standard unifié.
2
则
zé
3 emplois : (a) classificateur pour des textes courts (nouvelles, dictons, 一则消息) ; (b) conjonction de contraste "A…, (而) B 则…" = "quant à B, lui (en revanche)…" ; (c) conjonction de causalité (littéraire) = "dans ce cas, alors" (≈ 就 oral).
在第七课，我们学习了两则成语故事。
(a) Dans la leçon 7, nous avons étudié deux histoires de chéngyǔ (classifier = 则).
有的人家喜欢种草、养花、种竹子，有的人家则喜欢用大盆养金鱼。
(b) Certaines familles aiment planter des herbes, des fleurs, du bambou ; d'autres, elles, préfèrent élever des poissons rouges dans de grandes vasques.
猫享受独处的快乐，而狗则是希望和别人分享快乐。
(b) Les chats savourent la solitude ; les chiens, eux, cherchent à partager leur bonheur avec les autres.
北京的冬天，有风则寒，无风则暖。

3

wéi...suǒ...
Structure passive littéraire. "为 + agent + 所 + verbe" = "être V-é par (agent)". 为 signifie "par" (= 被). Structure figée, registre soutenu/écrit. Souvent avec négation "无不为……所" = "tout le monde est V-é par".

词语辨析 - Analyse comparative

通常 vs 常常

通常 (tōngcháng)常常 (chángcháng)
Points communsLes deux expriment une action qui se produit souvent de façon identique. Dans certaines phrases, ils sont interchangeables.
如：我通常/常常在学校外面那个超市买东西。
Differences 1. Insiste sur la régularité de l'action (schéma habituel).
如：有钱人家的，通常是由好几座四合院并列组成的。		 1. Insiste sur le nombre de répétitions (occurrence fréquente).
如：他成绩很好，常常受到表扬。
2. Adjectif — peut être épithète (定语).
如：我们通常的做法都是这样的。		 2. Adverbe uniquement — ne peut pas modifier une petite proposition ou être épithète.
如：他常常去上海出差，对上海很熟悉。
Astuce : 通常 = "normalement, habituellement" (régularité d'un schéma, peut être adj.). 常常 = "souvent, fréquemment" (nombre de fois, toujours adverbe). Test : si on peut remplacer par "normalement" → 通常 ; si par "souvent" → 常常.

背景分析 - Contexte : architecture et voyage

有人曾说："生活中不是缺少美，而是缺少发现美的眼睛。"美在生活中随处可见，建筑之美就是其中一个重要的组成部分。在世界各地旅行，我们可以欣赏到不同的建筑，从中也可以了解到各地不同的历史、文化、风俗等，具有传统特色的建筑是一个国家、一个城市的记忆，四合院就是中国建筑的一个代表作。许多外国或外地游客来到北京旅游时，都会专门去看看北京的胡同和四合院。
