⭐ Coup de cœur
Z2H · Zero to Hanzi z2h.cn/hanzi

L'outil le plus élégant pour décomposer les caractères chinois et comprendre comment ils sont construits. Un parcours "zéro vers hanzi" qui révèle la logique interne des sinogrammes — composants, étymologie, mnémoniques. Indispensable pour tout apprenant sérieux.

💡 Mon stack recommandé pour HSK 5

  • 📖Dictionnaire : Pleco (sur le téléphone) + Z2H (sur le web pour décomposer les caractères inconnus).
  • 🔁Vocabulaire : Anki avec un deck HSK 5 + nos modules "Apprentissage Commando" sur ce site.
  • 📚Lecture quotidienne : Du Chinese (15 min/jour à votre niveau) ou The Chairman's Bao pour les actualités.
  • 🎧Écoute passive : ChinesePod sur le trajet du matin + une chaîne YouTube native (Mandarin Corner).
  • ✍Hanzi : Skritter 3×/semaine sur les caractères de la leçon en cours.
  • 🎯Préparation HSK : 1 examen blanc complet (mode examen) tous les 15 jours pour tracker la progression.
  • 💬Oral : 1 à 2 cours italki par semaine — c'est ce qui fait vraiment décoller un niveau.
